Flagship Projects

Focusing on high-priority MSIL initiatives.

What is a Flagship Project?

Not all projects are created equal. High-priority, high-impact projects can be designated as Flagship Projects by checking the "Flagship" box during project creation or editing.

The Flagship view filters out the noise, providing executives with a clean, focused list of only the most critical initiatives currently underway.

Exporting Data

Executives often require formatted reports. The Flagship page includes a robust suite of export tools at the top right of the screen.

  • Export Snapshot: Takes an instant picture (PNG) of the table view.
  • Export PPTX: Automatically generates a branded PowerPoint presentation, creating a slide for each Flagship project.
  • Export Excel: Downloads a clean Excel workbook containing all data points for further analysis.
Pro Tip: Use the Columns Settings (⚙) button to show or hide specific columns before taking an Image Snapshot to customize what data is presented!
