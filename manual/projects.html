Adding & Editing Projects

Managing the MSIL data lifecycle.

Adding a New Project

To add a new project, click the + New Project button located at the top right of the Dashboard.

Because accountability is crucial, all fields in the initial project creation form are strictly mandatory. This ensures that every project enters the system with complete tracking parameters.

  • Basic Info: Project Name, Scope, Category, PIC, Division, Department, Section.
  • Budget: Estimated vs. Actual.
  • Contact Info: QA-BU Email for cross-functional communication.
  • Timestamps: Target and Actual completion dates for each IL Phase (IL1 through IL5).

Modifying a Project

Only users with appropriate permissions (Admin, DPM, SIC, TL, or the designated PIC) can modify a project.

Click the ✏️ Edit button next to a project row to open the modification modal.

New IL4 Fields: When a project reaches IL4 (UAT) or later, two new critical text fields appear at the bottom of the form: New Changes and Learnings. These are used to capture UAT feedback and institutional knowledge.
